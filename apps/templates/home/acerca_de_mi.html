Diogenes

Diogenes nace de la idea de tener un sistema de uso personal para poder cotizar mejor los trabajos "Freelance" que suelo encarar. Es po eso que necesitaba antes que nada saber cuales eran mis costos fijos para calcular el precio de los trabajos.

En esta primera instancia, a modo de MVP, el sistema permite ingresar los gastos ya categorizados para luego poder agruparlos, y en base a eso tener una informacion mas detallada de los gastos.

Esta realizado en Django nativo y html 5, css 3, javascript, jquery, bootstrap, fontawesome, etc. como frontend. Para el dashboard, se utiliza librearias JS como sparkline, chartjs, circle progress, etc.

Dioegenes tambien cuenta con una app de servicio de mensajeria para que los usuarios puedan enviarse mensajes entre si (user to user)
